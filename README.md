# Realzer
## Real Estate Website App with rental home and sale properties.
### Technologies:
- React
- Next.js
- Chakra UI
- Tailwind CSS
- REST API
- Axios
- Node.js

**Developed User Interface using Chakra UI.**

**Static Generation and Server Side Rendering with Next.js**

**Real Estate Data fetched from Rapid API.**
